# Assets

Open Pixel currently uses lightweight placeholder assets so the repo is submission-ready without external licensing risk.

## Included

- `assets/open-pixel-logo.svg` — generated logo placeholder.
- `assets/open-pixel-demo.gif` — generated animated README demo placeholder.
- `assets/demo-placeholder.svg` — static fallback.
- `apps/game/public/audio/collect.wav` — generated UI collect sound.
- `apps/game/public/audio/quest-complete.wav` — generated quest complete sound.
- `apps/game/public/audio/click.wav` — generated UI click sound.
- RPG-JS starter tiles/sprites under `apps/game/src/tiled` and `apps/game/public/spritesheets`.

## Before final submission

Replace placeholders with original or properly licensed art:

- final logo
- gameplay GIF or video thumbnail
- NPC portrait / cover image
- thumbnail/cover for Zero Cup project form
- optional ambient loop

## Licensing rule

Do not copy Pixels.xyz, Sunflower Land, Kaetram, or commercial game assets. Use original assets, CC0 assets, or assets with explicit commercial/hackathon-friendly license.
