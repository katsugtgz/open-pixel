# Submission Draft

## Title

Open Pixel

## Summary

Guest-first cozy Web3 pixel RPG where players do resource-village work, earn off-chain progress, and can optionally prove completion with a readable wallet signature.

## Description

Open Pixel is a browser-based pixel RPG prototype for Zero Cup 2026. Players enter as guests, move through a cozy resource village, perform actions like farming, chopping, mining, crafting, or preparing goods, then fulfill a small board/workstation request to earn off-chain progress and a Guest Badge.

Wallet verification is optional and intentionally safe: the claim page only asks for a readable `personal_sign` message after completion. It never asks for approvals, swaps, transactions, permits, or contract calls.

The project explores how Web3 identity/proof can be added to a game without turning the core loop into token speculation.

## Safety Line
