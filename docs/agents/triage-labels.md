# Triage Labels

Use these labels for agent workflow:

- `needs-triage`: incoming issue needs clarification and routing.
- `needs-info`: human input is required before implementation can proceed.
- `ready-for-agent`: issue is sufficiently scoped for an implementation agent.
- `ready-for-human`: agent work is complete and needs human review.

GitHub default labels may still be used for issue type, such as `enhancement`, `bug`, or `documentation`.
