# Domain Notes

Open Pixel is a guest-first RPG-JS pixel quest prototype. It uses readable off-chain progression and optional wallet proof; it must not introduce token economy, real-money trading, approvals, transactions, staking, swaps, or marketplace mechanics.

For gameplay planning, use the glossary in the root context document:

- **Cozy Resource-Village Loop**
- **Mechanic Grammar**
- **Layout Grammar**
- **Layout Composition**

Gameplay work must preserve the real RPG-JS canvas and should improve the in-engine map, events, assets, and quest interactions rather than replacing them with DOM fallbacks.
