# Visual style

Open Pixel should feel adjacent to cozy browser worlds like Pixels.xyz without copying their assets.

## Direction

- Bright pixel-web landing, not generic AI SaaS dark gradient.
- Chunky buttons, thick dark outlines, playful rounded panels.
- Cozy field/world preview: sky, clouds, grass tiles, paths, crystals, NPC, player.
- Web3 proof UI should feel like an in-game receipt, not a DeFi dashboard.

## Typography

- Headings/brand: pixel-style font stack (`Silkscreen`, monospace fallback).
- Body: readable sans (`Inter`, system fallback).
- Avoid tiny cyberpunk text; keep contest reviewers moving fast.

## Components

- Buttons: lime or pink, dark 4px border, blocky shadow.
- Cards: dark purple panel, cream/lime accents, 5px border.
- Badges: rounded, cream fill, dark outline.
- Hero art: CSS-generated pixel island so no external art license risk.

## Guardrails

- Do not copy Pixels.xyz images, fonts, sprites, logos, or exact layouts.
- Use reference-level palette/mood only.
- Keep wallet copy explicit: no tx, no gas, no approvals, no swap, no permit.
